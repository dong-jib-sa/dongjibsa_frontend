## 프로젝트 소개 

2023 서울 우먼테크 해커톤에서 '지속가능한 미래 도시'에 대한 주제로 청년 1인 가구의 지속가능한 한 끼를 위해 필요한 양의 식재료만 구매하여 식생활 제로웨이스트를 실천할 수 있게 하는 '동네 기반 O2O 식재료 공동구매 커뮤니티 플랫폼'입니다.
